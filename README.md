# Realtor Partner Application Form

3-step form for recruiting new partner realtors into the Service First Partner Program. Submissions are sent to a Formspree (or similar) endpoint.

**Before deploying:** open `src/App.jsx` and replace the placeholder `SUBMISSION_ENDPOINT` near the top of the file with a real Formspree URL.

Built with Vite + React + Tailwind. Deploys to Vercel automatically.
